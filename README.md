# Arrestra — Landing + Intake (Next.js)

A production-ready landing page with an intake form that emails submissions.

## Quick start
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Environment
Create `.env.local` with:
```
RESEND_API_KEY=your_resend_key_here
LEAD_NOTIFY_EMAIL=intake@arrestra.com
```

On AWS Amplify, add these in **Hosting → Environment variables**, then redeploy.

## Build
```bash
npm run build
npm start
```
